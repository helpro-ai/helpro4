# Helpro

Helpro is a demo marketplace for everyday help inspired by peer-to-peer services. It includes:

- React + Vite front-end with marketing pages, dashboard, requests/bookings, messaging, and profile screens.
- AI assistant chat widget with mock backend responses.
- Minimal Express mock API served at `/api/ai/chat`.

## Getting started

```bash
npm install
npm run dev         # start Vite dev server
npm run dev:api     # start Express mock API
npm run dev:full    # start API + Vite together
```

## Building for production

```bash
npm run build
```
